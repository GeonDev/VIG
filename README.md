# VIG(Visual Inspiration Gallery)
사용자 맞춤형 이미지 검색 기반 포트폴리오(핀터레스트 & 비헨스) 클론 프로젝트 입니다. 

## 적용 내용
#### 1. java 1.8
#### 2. Spring 5.0
#### 3. MariaDB 10.4
#### 4. Mybatis 3.2.1
#### 5. JSTL/EL 적용

## 기타 적용 
###### * apache tomcat 8.5(servlet 3.1) 사용
###### * bootstrap 4.0/MDB 적용
###### * Ajax/RestFul 서버 적용
###### * Web socket(실시간 알람) 적용
###### * google API(login, VISION, TRANS) 적용 
###### * 결제(I'm port) 적용
